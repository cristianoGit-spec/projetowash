# ÍCONES PWA

Para funcionar como PWA completo, você precisa adicionar ícones nas seguintes dimensões:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Como Criar os Ícones

### Opção 1: Ferramenta Online (Recomendado)
Use o site: https://www.pwabuilder.com/imageGenerator

1. Faça upload de uma logo (quadrada, 512x512px ou maior)
2. Clique em "Generate"
3. Baixe o pacote ZIP
4. Extraia os ícones para esta pasta

### Opção 2: Usar uma Logo Temporária

Enquanto não tem os ícones, você pode:
1. Criar uma imagem quadrada simples (ex: com as letras "QC" de Quatro Cantos)
2. Redimensionar para cada tamanho listado acima
3. Salvar nesta pasta

### Opção 3: Usar Favicon Generator
Use: https://realfavicongenerator.net/

Este site gera automaticamente todos os tamanhos necessários.

## Dica de Design

Crie um ícone simples e reconhecível:
- Use as cores do sistema (azul #2563eb)
- Pode ser um símbolo de estoque/armazém
- Ou as iniciais "EC" em fonte grande e legível
- Fundo sólido ou gradiente
- Evite detalhes muito pequenos

## Ferramenta de Redimensionamento

Se você tem uma imagem grande, use:
- **Online:** https://imageresizer.com/
- **Windows:** Paint, GIMP
- **Mac:** Preview
- **Linux:** GIMP, ImageMagick

## Cores Sugeridas

Baseado no design do sistema:
- Cor principal: #2563eb (azul)
- Cor secundária: #1d4ed8 (azul escuro)
- Fundo: Branco ou gradiente azul

## Nota

O sistema funcionará sem os ícones, mas eles são importantes para:
- Instalação como PWA
- Aparência profissional
- Reconhecimento na tela inicial
